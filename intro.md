Đây là một dự án website thương mại điện tử bán cây cảnh, chậu cảnh và phụ kiện decor trực tuyến có tên "Thế Giới Cây Xanh XWORLD". Dự án được xây dựng bằng Laravel framework với đầy đủ các tính năng quản lý sản phẩm, đơn hàng, nội dung và hệ thống admin. Ngoài ra còn tích hợp các dịch vụ thanh toán (PayOS), vận chuyển (GHN) và các tính năng SEO tối ưu.

### Các Chức Năng Chính

* Quản lý sản phẩm (CRUD, hình ảnh, danh mục phân cấp, tồn kho, Flash Sale)
* Quản lý đơn hàng (tạo từ giỏ hàng, tích hợp GHN shipping, tích hợp PayOS payment, theo dõi trạng thái, in hóa đơn PDF)
* Quản lý nội dung (blog/bài viết, tags, bình luận, đánh giá, banner quảng cáo, newsletter)
* Hệ thống quản lý tài khoản, phân quyền, thùng rác (soft delete), sao lưu & khôi phục database, sitemap tự động
* Tối ưu hóa SEO, bảo mật (CSRF, XSS, SQL injection protection, rate limiting, password hashing)

### Công Nghệ Sử Dụng

* **Backend** : PHP >= 8.2, Laravel framework, Composer >= 2.0
* **Frontend** : Blade templates, JavaScript, CSS, Node.js >= 18.x
* **Database** : MySQL >= 8.0 hoặc MariaDB >= 10.3
* **Web Server** : Nginx hoặc Apache với mod_rewrite
* **Build Tool** : Vite
* **Ngôn ngữ** : Blade (54.3%), PHP (32.6%), JavaScript (7.0%), CSS (6.1%)
